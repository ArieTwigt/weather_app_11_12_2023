# Weather prediction module


# Installation

## Install virtuelenv

`pip install virtualenv`

## Create a virtualenv

`virtualenv venv --python=python3.12`

or

`python -m venv venv`

## Activate the virtual environment

**Mac**

`. venv/bin/activate`


**Windows**

`venv\Scripts\activate`


## Install the modules 

`pip install -r requirements.txt`


## Use the module

See usage with:

`python main_weather.py -h`
